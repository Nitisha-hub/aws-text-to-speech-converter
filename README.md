# AWS Text-to-Speech Converter            

## Project Overview
AWS Text-to-Speech Converter is a cloud-based web application developed using Python, Flask, and AWS cloud services. The application converts user-provided text into natural-sounding speech using Amazon Polly and securely stores generated audio files using Amazon S3.

This project demonstrates the practical implementation of:  
- Cloud Computing
- Speech Synthesis Technology
- AWS Service Integration
- Backend Web Development

---

## Features
- Convert text into speech
- Natural-sounding voice generation
- Audio file generation and playback
- Secure cloud storage using Amazon S3
- User-friendly web interface
- Fast and scalable cloud-based processing

## Project Workflow
1. User enters text in the web application  
2. Flask backend receives the text input  
3. Amazon Polly converts text into speech  
4. Generated audio file is stored in Amazon S3  
5. Audio output is provided to the user
